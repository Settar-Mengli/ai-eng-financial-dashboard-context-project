# Project Brief

## Purpose

This repository is a financial metrics dashboard. `README.md` describes it as a React + TypeScript frontend with a FastAPI backend.

## Current Product Shape

- Frontend service: Vite React app served on `http://localhost:5173`.
- Backend service: FastAPI app served on `http://localhost:8000`.
- API documentation: `http://localhost:8000/docs`.
- Docker Compose builds and runs separate `frontend` and `backend` services.

## Project 1 Scope

Project 1 is documentation and context-building only:

- Understand the inherited codebase from repository evidence.
- Document good and risky engineering practices.
- Create repository rules in `.agents/rules`.
- Initialize the memory bank in `memory-bank`.
- Preserve application behavior.

## Out Of Scope

- Frontend feature work.
- Backend feature work.
- Runtime, Docker, dependency, package, or test changes.
- Refactors or bug fixes that belong to later project phases.
